# 音伴 — HTML 原型说明

> **本目录为设计验证原型，非生产前端源码。** 生产实现使用 React + TypeScript + Vite + Ant Design。

## 打开方式

在浏览器中打开 `index.html`（推荐 1440×900 视口）。

## 布局参考

- 用户参考图：三栏播放工作台（左推荐 / 中黑胶播放器 / 右 AI 聊天）
- 首页：居中 Hero 心情输入 + 右侧 AI 侧栏
- 顶栏导航：首页 / 歌单 / 谱库 / 我的

## 页面清单

| 原型区域 | PRD 页面 | 说明 |
|---------|---------|------|
| 偏好弹层 | P01 | 首次访问显示 |
| 首页 Hero + AI 侧栏 | P02 | 心情输入、快捷标签 |
| 三栏播放工作台 | P02b | 内嵌播放、推荐列表 |
| 谱面抽屉 | P03 | 吉他/尤克里里 Tab |
| 歌单列表/详情 | P04/P05 | 顶栏「歌单」进入 |
| 我的 | P06 | 顶栏「我的」进入 |

## 交互动效

- 首页元素错落入场、侧栏滑入
- 发送心情 → AI 打字指示器 → 推荐列表渐入
- 播放页三栏入场、黑胶浮动 + 旋转、进度条流光
- 按钮涟漪、卡片 hover 上浮、切歌 meta 动画
- 支持 `prefers-reduced-motion` 降级

## 可点击路径（至少 3 条）

1. 首页输入心情 → 发送 → 进入三栏播放页
2. 点击推荐歌曲 / 播放按钮 → 黑胶播放器开始旋转
3. 「吉他谱」「尤克里里谱」→ 右侧谱面抽屉
4. 顶栏「歌单」→ 歌单详情 → 点击歌曲回到播放页

## 需真实 API 的位置

| 功能 | API |
|------|-----|
| 情绪对话 | `POST /api/v1/chat` |
| 荐歌 | `POST /api/v1/chat` + 推荐卡片 |
| 播放地址 | `GET /api/v1/songs/{id}/play-url` |
| 谱面 | `GET /api/v1/songs/{id}/score` |
| 歌单 CRUD | `/api/v1/playlists/*` |

## 生产实现注意

- 黑胶视觉仅为原型展示，生产可用简化封面 + 标准播放器控件
- 三栏布局在窄屏需降级为单栏 + 底部迷你播放器
- 游客模式顶栏显示徽章，无登录入口
